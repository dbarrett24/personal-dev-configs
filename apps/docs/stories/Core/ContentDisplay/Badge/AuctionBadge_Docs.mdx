## Overview

The AuctionBadge component displays the live or paused status of an auction with viewer count. This component shows a badge with auction status (LIVE/PAUSED/CLOSING), optional video camera icon, viewer count with dot separator, and appropriate state indicators. Built with Hammer UI components for consistency with the design system.

### Props

- `className`: Optional className for custom styling and positioning
- `hasVideo`: Whether the auction has video streaming capabilities
- `isClosing`: Whether the auction is currently closing
- `isPaused`: Whether the auction is currently paused
- `viewers`: Number of viewers currently watching the auction

### Code Example

This is a basic example of the component with default settings:

```tsx
import { AuctionBadge } from '@dbarrett24/core-components/Badge/AuctionBadge';

const ExampleComponent = () => {
    return (
        <AuctionBadge
            hasVideo={false}
            isClosing={false}
            isPaused={false}
            viewers={126}
        />
    );
};
```

## Visual States

The AuctionBadge displays different visual states based on the auction status:

- **LIVE**: Shows "LIVE" text with viewer count and dot separator
- **LIVE with Video**: Includes a video camera icon before the "LIVE" text
- **PAUSED**: Shows pause icon and "PAUSED" text with viewer count
- **CLOSING**: Shows "CLOSING" text without icons or viewer count
