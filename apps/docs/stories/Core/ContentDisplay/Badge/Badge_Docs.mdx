## Overview

The badge component is effectively a styled `span`.
It is used to display a small amount of information, such as "live", in a visually appealing way.
It is often placed absolutely.

<Canvas
    of={BadgeStories.DefaultBadge}
    sourceState="none"
/>

<Controls of={BadgeStories.DefaultBadge} />

### Basic Usage

This shows how you could overlay it over a (mocked) image.

import { ExampleComponent as Basic } from './Basic.example.js';

### With icons

You can place an icon inside the component with either the `iconLeft` or `iconRight` props.

import { ExampleComponent as WithIcon } from './WithIcon.example.js';
import WithIconText from './WithIcon.example.tsx?raw';

<ElevatedSection>
    <WithIcon />
</ElevatedSection>
